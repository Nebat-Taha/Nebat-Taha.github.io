AI-Driven Infrastructure Orchestrator

Bridging the gap between Project Management (Jira) and Version Control (GitHub) using Local LLMs.

The Workflow
  • Trigger: Manual Jira Automation rule (triggered via UI button).
  • Middleware: FastAPI receives the webhook payload (Summary, Key, Labels).
  • Contextual Prompting: System matches Jira labels to local markdown prompts (e.g., `terraform.md`).
  • Intelligence: Local LLM (Ollama) generates the code/fix based on the summary.
  • Action: Automatically creates a GitHub branch and PR, or updates an existing one with comments.
  • Feedback Loop: Posts the PR link back to the Jira ticket for seamless tracking.
Tech Stack & Security

Built with Python and FastAPI. Security is managed via .env configurations for GitHub tokens and Jira API credentials, ensuring no sensitive data is hardcoded.
