02. Prometheus & Service Discovery

Architectural Goal: Automated Scraping

Implemented AWS EC2 Service Discovery in Prometheus. By targeting specific EC2 tags, Prometheus automatically discovers new application nodes as they scale, eliminating the need for manual configuration updates.

03. Visualization: Unified Dashboards

Architectural Goal: Contextual Awareness

Engineered multi-tenant dashboards in Grafana that correlate infrastructure health with application error logs. This creates a "Single Pane of Glass" for rapid incident response.

04. Intelligent Alertmanager Routing

Architectural Goal: Reducing Fatigue

Configured Alertmanager to route high-priority issues to PagerDuty and low-priority warnings to Slack, using grouping logic to prevent alert storms during mass outages.
